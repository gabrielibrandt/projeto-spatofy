# 🎵 Spatofy

Spatofy é um projeto de recomendação de músicas desenvolvido com foco educacional. A ideia principal do projeto foi disponibilizar um código-base já estruturado para alunos que estavam aprendendo os fundamentos do desenvolvimento web.

O projeto foi utilizado como apoio prático durante os estudos de:

* HTML
* CSS
* JavaScript
* Git
* GitHub

Além de servir como exemplo para organização de arquivos, estrutura de páginas e versionamento de código.

---

# 📚 Objetivo do Projeto

O Spatofy foi criado para ajudar alunos iniciantes a compreenderem como um projeto front-end pode ser organizado na prática.

Com ele, os estudantes puderam:

* Entender a estrutura básica de um site
* Trabalhar com estilização usando CSS
* Adicionar interações simples com JavaScript
* Aprender conceitos de versionamento com Git
* Publicar e compartilhar projetos utilizando GitHub

O projeto também funciona como um ponto de partida para futuras melhorias e personalizações.

# 💻 Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone https://github.com/gabrielibrandt/projeto-spatofy.git
```

## 2. Acesse a pasta do projeto

```bash
cd spatofy
```

## 3. Abra o arquivo `index.html`

Você pode abrir diretamente no navegador ou utilizar uma extensão como:

* Live Server (VS Code)

## 4. Ou então acessar pelo GitHub Pages

https://gabrielibrandt.github.io/projeto-spatofy/
